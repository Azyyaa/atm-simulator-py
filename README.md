# ATM Simulator

A console-based ATM simulator developed in Python.

## Features

* PIN verification
* Change PIN
* Deposit funds
* Withdraw funds
* Check account balance
* Limited PIN attempts and card blocking

## Technologies

* Python
* File handling

## How to Run

1. Download the project files.
2. Open the project in PyCharm or another Python IDE.
3. Run the main Python file.

## Project Structure

* `main.py` – main program logic
* `pin.txt` – stores the PIN code
* `balance.txt` – stores the account balance
